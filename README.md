# 📚 Sobre o Repositório

Este repositório foi criado para acompanhar minha evolução na programação utilizando a linguagem Java.

Aqui reúno exercícios, desafios e pequenos projetos desenvolvidos durante meus estudos de lógica de programação e algoritmos.
